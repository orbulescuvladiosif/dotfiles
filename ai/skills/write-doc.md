# Doc Writing

## Principles

- Open with what the thing is, not what it does
- Lead with the conclusion — state it, then support it
- Stay at the right altitude — cut anything that belongs in a different doc or a later phase
- One idea per section — if a section needs two, split it
- Use formatting only when structure demands it — hierarchy through weight, not decoration

## Language

- **Abbreviations:** first use = full term followed by abbreviation in parentheses — e.g. "Product Vision Document (PVD)"; abbreviation alone from that point on

## Visual Template (Word)

Default for all Word documents produced under these conventions. Use this unless another skill specifies a different template or you're given reference material to match.

### Fonts

| Role | Font | Size | Style | Color |
|------|------|------|-------|-------|
| Title | Aptos Display | 28pt | Bold | `#1F2D3D` |
| Heading 1 | Aptos Display | 20pt | Bold | `#1F2D3D` |
| Heading 2 | Aptos | 14pt | Bold | `#1F2D3D` |
| Heading 3 | Aptos | 12pt | Bold | `#2D5F8A` |
| Body | Aptos | 11pt | Regular | `#1A1A1A` |
| Caption / Label | Aptos | 9pt | Regular | `#595959` |

Fallback if Aptos unavailable: Calibri (same sizes).

### Color Palette

| Name | Hex | Use |
|------|-----|-----|
| Navy | `#1F2D3D` | Headings, title, strong emphasis |
| Blue | `#2D5F8A` | H3, links, accent elements |
| Body text | `#1A1A1A` | All body copy |
| Muted | `#595959` | Captions, footers, labels |
| Callout bg | `#EBF3FB` | Info boxes, highlighted sections |
| Rule / divider | `#C5D9E8` | Horizontal rules, table borders |

### Spacing & Layout

- **Margins:** 2.54 cm (1 in) all sides
- **Body line spacing:** 1.15, 6pt after paragraph
- **Before H1:** 24pt, after: 8pt
- **Before H2:** 18pt, after: 6pt
- **Before H3:** 12pt, after: 4pt
- **Page size:** A4 (default); Letter if recipient is North American

### Page Structure

1. **Cover** (title page): document title centered (Title style), subtitle, author, date, version
2. **Table of contents** — include for docs longer than 3 pages
3. **Body** — H1 → H2 → H3 hierarchy; never skip a level
4. **Appendices** — if referenced from body; labeled Appendix A, B…

### Header / Footer

- **Header:** document title (left) | section name (right)
- **Footer:** classification or "Internal" label (left) | page number (right, format: `Page X of Y`)
- Header/footer font: Aptos 9pt, Muted (`#595959`)
