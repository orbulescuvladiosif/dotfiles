# SDLC orchestrator

**2026-06-16**

Original design: gates inside individual skills — no lifecycle orchestrator.

Added `sdlc` — Understand → Plan → Build → Check → Refine. User gates at understanding, plan approval, and check. Build modes: Autonomous, Checkpoint, MITM. Visible AI gates for claude-mem, self-review, pre-quality.

Bridges isolated skill gates and end-to-end non-trivial work.
